项目简介

基于 Hexo 静态博客框架，结合 GitHub Pages 实现免费托管，使用阿里云完成域名解析与 HTTPS 配置，并通过图床服务管理图片资源。适合技术爱好者快速搭建高性能、低成本的个人博客。

技术栈

  • 博客框架：Hexo（快速生成静态网站）
  • 托管平台：GitHub Pages（免费静态托管）
  • 云服务：阿里云（域名注册、DNS 解析、SSL 证书）
  • 图床：七牛云 / 阿里云 OSS/Imgur（推荐 PicGo 管理图片）
  • 版本控制：Git（管理博客源码与部署文件）

环境要求

  1. 必备工具：
    • Node.js（v14+，Hexo 运行环境）
    • Git（代码管理）
    • Docker（可选，容器化部署）
  2. 阿里云配置：
    • 注册阿里云账号，完成实名认证
    • 购买域名（如 yourname.com）
    • 开通对象存储服务（OSS，用于图床）

阿里云配置：域名与 HTTPS

1. 域名解析

  1. 登录阿里云控制台，进入 域名管理
  2. 添加 DNS 解析记录：
    • 类型：A，主机记录：@，解析目标：185.1X9.XX8.153（GitHub Pages IP，需定期更新）
    • 类型：CNAME，主机记录：www，解析目标：YourUsername.github.io

2. 配置 HTTPS（SSL 证书）

  1. 在阿里云 SSL 证书 页面申请免费证书（如 Let’s Encrypt）
  2. 下载证书文件，在 GitHub Pages 仓库 Settings → Pages 中启用 HTTPS
  3. 在 Hexo 项目根目录创建 source/CNAME 文件，写入域名（如 yourname.com）
